# gruber-darker-sublime
a port of [gruber-darker](https://github.com/rexim/gruber-darker-theme) to sublime

just copy gruber-darker.sublime-color-scheme into your ~/.config/sublime-text/Packages/User directory and it will
be available when selecting a color scheme.

it's tested on python and go for now, if you've managed to get it to look the same as emacs for other languages submit a pull request but please don't break go/python this shit is a pain to get it right.
